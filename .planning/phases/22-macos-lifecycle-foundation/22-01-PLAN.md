4. Seven H2 stage sections, each with four H3 subsections:
   - `## Stage 1: ABM Device Registration`
     - `### What the Admin Sees` - ABM portal view (business.apple.com > Devices > Assign to MDM Server)
     - `### What Happens` - Serial number assignment to MDM server, OEM pre-assignment possibility
     - `### Behind the Scenes` - MDM server assignment, serial number as identity (unlike Windows hardware hash). Link to [ABM](../_glossary-macos.md#abm) glossary entry.
     - `### Watch Out For` - Device not assigned to correct MDM server; non-ABM-linked reseller; reseller forgot device transfer. Content from 22-RESEARCH.md Stage 1.

   - `## Stage 2: ADE Token Sync`
     - Same 4 H3 subsections. Key content: .p7m token, annual renewal, 24-hour auto-sync, 15-minute manual sync rate limit, 7-day full sync cooldown. Link to [ABM Token](../_glossary-macos.md#abm-token). Admin portal: Intune admin center > Devices > Enrollment > Apple tab > Enrollment program tokens.

   - `## Stage 3: Enrollment Profile Assignment`
     - Same 4 H3 subsections. Key content: Profile defines user affinity, auth method, Await Configuration on/off, Setup Assistant screens. Profile must be assigned BEFORE device powers on. "Setup Assistant with modern authentication" is the default as of late 2024. Link to [Setup Assistant](../_glossary-macos.md#setup-assistant).

   - `## Stage 4: Setup Assistant`
     - Same 4 H3 subsections. Key content: Device contacts deviceenrollment.apple.com, iprofiles.apple.com, mdmenrollment.apple.com. MDM enrollment profile installs silently. Configurable screens (Apple ID, Siri, Privacy can be hidden). Modern auth = Entra credential sign-in during Setup Assistant. ACME certificate on macOS 13.1+. Link to [Setup Assistant](../_glossary-macos.md#setup-assistant) and [ADE](../_glossary-macos.md#ade).

   - `## Stage 5: Await Configuration`
     - Same 4 H3 subsections. Key content: "Awaiting final configuration" locked screen. Only fires when enrollment profile has Await Configuration = Yes (default since late 2024). No Intune-enforced timeout. Microsoft testing shows ~15 minutes typical. Supported macOS 10.11+. Does NOT fire on re-enrollment. Link to [Await Configuration](../_glossary-macos.md#await-configuration).

   - `## Stage 6: Company Portal Sign-In`
     - Same 4 H3 subsections. Key content: Only for "Enroll with User Affinity" + modern auth. Company Portal must be deployed as required app. Sign-in registers device with Entra ID. Without sign-in, Conditional Access blocks protected resources. Userless enrollments skip this stage. Link to [VPP](../_glossary-macos.md#vpp) if mentioning app deployment method.

   - `## Stage 7: Desktop and Ongoing MDM`
     - Same 4 H3 subsections. Key content: Two parallel management channels (Apple MDM via APNs for config profiles, IME for shell scripts/apps). IME agent at `/Library/Intune/Microsoft Intune Agent.app`. APNs certificate annual renewal (separate from ADE token). FileVault, firewall, Gatekeeper, compliance via MDM profiles. LAPS rotation if configured.
