**Title:** `# L2 Log Collection Guide`

**Context section:** Brief explanation that this guide is the prerequisite for all L2 investigation runbooks. Explains the "gather everything first, then investigate" workflow (per D-06). First mention of glossary terms (ESP, OOBE, MDM) linked to `../_glossary.md#term`.

**Section: Diagnostic Package (mdmdiagnosticstool.exe)**
Include the exact command:
```powershell
mdmdiagnosticstool.exe -area Autopilot -cab -out C:\Temp\AutopilotDiag
```
Note that the `.cab` file is the critical artifact for Microsoft Premier Support sharing.

**Section: One-Command Alternative (Get-AutopilotLogs)**
```powershell
$logPath = Get-AutopilotLogs -OutputPath "C:\Temp\AutopilotDiag\$env:COMPUTERNAME"
```
Link to `[Get-AutopilotLogs](../reference/powershell-ref.md#get-autopilotlogs)`.

**Section: Event Viewer Exports**
4 required exports with PowerShell `wevtutil epl` commands using the naming convention:
```powershell
$serial = (Get-WmiObject -Class Win32_BIOS).SerialNumber
$date = Get-Date -Format "yyyy-MM-dd"
$prefix = "$date`_$serial"

wevtutil epl "Microsoft-Windows-DeviceManagement-Enterprise-Diagnostics-Provider/Admin" "$prefix`_mdm-admin.evtx"
wevtutil epl "Microsoft-Windows-Provisioning-Diagnostics-Provider/Admin" "$prefix`_provisioning.evtx"
wevtutil epl "Microsoft-Windows-AAD/Operational" "$prefix`_aad.evtx"
wevtutil epl "Microsoft-Windows-User Device Registration/Admin" "$prefix`_user-device-reg.evtx"
```

**Section: Registry Snapshot Export**
```powershell
reg export "HKLM\SOFTWARE\Microsoft\Windows\Autopilot\EnrollmentStatusTracking" "$prefix`_esp-registry.reg" /y
```
Link to [registry-paths.md](../reference/registry-paths.md) for ESP tracking key descriptions. Note: live registry reads are for devices still in-flight on ESP; post-failure analysis uses the cab.

**Section: Artifact Naming Convention**
Table with standard naming pattern for Microsoft support sharing:
| Artifact | Filename Pattern |
|----------|-----------------|
| Diagnostic cab | `YYYY-MM-DD_SerialNumber_autopilot-diag.cab` |
| MDM Admin log | `YYYY-MM-DD_SerialNumber_mdm-admin.evtx` |
| Provisioning log | `YYYY-MM-DD_SerialNumber_provisioning.evtx` |
| AAD log | `YYYY-MM-DD_SerialNumber_aad.evtx` |
| User Device Registration | `YYYY-MM-DD_SerialNumber_user-device-reg.evtx` |
| ESP registry snapshot | `YYYY-MM-DD_SerialNumber_esp-registry.reg` |

**Section: Tool References**
- Link to [PowerShell Reference](../reference/powershell-ref.md) — Get-AutopilotLogs function
